# Boas-vindas ao repositório do projeto Trivia!

Trata-se de um jogo de perguntas e respostas onde se tem 30 segundos para responder cada pergunta, num total de 5 perguntas.
Após, o usuário é direcionado para a tela de score, onde o texto mostrado depende do número de acertos.
O placar depende da dificuldade da pergunta e da velocidade da resposta. As perguntas vêm da api https://opentdb.com/ .
O usuário pode:
  - Logar no jogo e, se o email tiver cadastro no site [Gravatar](https://pt.gravatar.com/), ter sua foto associada ao perfil da pessoa usuária.
  - Acessar a página referente ao jogo, onde se deverá escolher uma das respostas disponíveis para cada uma das perguntas apresentadas.
A resposta deve ser marcada antes do contador de tempo chegar a zero, caso contrário a resposta deverá ser considerada errada.
  - Visualizar a página de ranking, se quiser, ao final de cada jogo.
  - Configurar algumas opções para o jogo em uma tela de configuração acessível a partir do cabeçalho do app.
